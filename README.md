\## Monetary Units



The Honey (HNY) network uses a fixed 18-decimal monetary model.



\- \*\*Base unit:\*\* CONE  

\- \*\*Display unit:\*\* HONEY (HNY)



\### Conversion





\### Ledger Application Behavior



\- All transaction signing is performed using \*\*CONES\*\*

\- Ledger UI displays values formatted as \*\*HNY\*\*

\- No floating-point arithmetic is used at any stage

\- Precision is never truncated during signing



This design mirrors best practices used by Ethereum and Cosmos-SDK Ledger applications.
